# Student Management System

A desktop-based Student Management System developed using Python, Tkinter, and MySQL.

## Features

* Login authentication
* Add student records
* Update student details
* Delete student records
* Search student information
* Export records to CSV
* MySQL database integration

## Technologies Used

* Python
* Tkinter
* MySQL
* PyMySQL
* Pandas
* ttkthemes
* Pillow

## Installation

Install the required packages:

pip install pymysql pandas ttkthemes pillow

## How to Run

1. Make sure MySQL Server is running.
2. Open the project folder.
3. Run:

python login.py

4. Enter your MySQL username and password.
5. The application will automatically create the database and tables if they do not exist.
